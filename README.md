# Spring React E-commerce Platform

Full-stack e-commerce application built with Spring Boot and React.

## Features

- User authentication and registration with JWT
- Account verification via email (Brevo API)
- MySQL database for persistent storage
- RESTful APIs for user and product management
- (In progress) Shopping cart and order tracking

## Tech Stack

- **Backend:** Java, Spring Boot, Spring Security, JWT, MySQL, Brevo API
- **Frontend:** React, React Router, Axios
